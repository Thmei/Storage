What is Pikture?

Pikture is a Website designed to allow artists to sell their media with various licensing options.

What is special about Pikture?

What is special about Pikture is that customers can post art requests which artists can then accept.

Do I need to log in to use this site?

Yes. To be able to upload an art, you need to be registered as an Artist user. In addition, to buy, you need to be registered as a Customer user.

What type of licensing is there?

There are three kinds of licenses:

  1. Unlimited License - A license to print and use the image on the Web
  2. Printed License - A license to get a printed copy of the art.
  3. Web License - A license to use the image on the web.

What types of accounts are available?

There are three kinds of accounts:

  1. Artist - You will be able to upload and sell the arts that you make depending on what license.
  2. Customer - You will be able to purchase a license of an art
  3. Admin - You will be responsible for the database of the site.

How can I search for specific images on the website?

You can browse images by categories or search by free text – both options are available.

What kind of statistics is available to the artists using the website?

An artist can see statistics on number of clicks for all of his/her images. The statistics are presented in the form of a visual graph making it really easy to see how popular the artist images are.
